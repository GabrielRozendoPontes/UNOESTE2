# Forma Normal Disjuntiva
### a)
$Q(a, b, c) = a * b+a * c$

$= \ a * b * (c + c') + a * c * (b + b')$

$= \ a * b * c + a * b * c' + a * c * b + a * c * b'$

$= \ a * b * c + a * b * c' + a * b' * c$
---
### b)
$Q(a, b) = a \to b$

$= \ a'+b$

$= \ a' * (b+b') + b * (a+a')$

$= \ a' * b + a' * b' + b * a + b * a'$

$= \ a' * b' + b * a + b * a'$
